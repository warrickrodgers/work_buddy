## 6. Supporting Process Frameworks
These provide structure and rigor around human-centered improvement:

### Lean Six Sigma (DMAIC)
- Define → Measure → Analyze → Improve → Control  
Use for operational excellence while maintaining human engagement.

### PDCA Cycle (Plan–Do–Check–Act)
- Continuous, incremental improvement cycle.
- Works best when paired with *team reflection sessions* (learning rituals).

### Kaizen Philosophy
- Daily, small improvements with shared responsibility.
- Built on trust, transparency, and mutual respect.

### Kotter’s 8-Step Change Model
- Especially powerful when paired with *storytelling* and *symbolic leadership*.
